-------------------------------
---- Телефонный справочник ----
-------------------------------

Краткое описание.

Данные справочника хранятся в файле csv_phonebook.csv.
Данные из справочника не удаляются (и не перезаписываются).
Данные только добовляются в файл csv_phonebook.csv.
Удаленные и обновленные записи (версия записи до обновления) помечаются как "не активные" (не видимые для пользователя).
Список row_id не активных записей хранится в файле csv_phonebook_activity_id.csv (такие записи не видимы в справочнике).

Файл csv_phonebook_activity_id.csv можно использовать как файл логирования для удаленных и обновленных записей.

Список полей:
row_id	- уникальный идентификатор записи (не видим для пользователя)
id	- уникальный идентификатор активной записи
surname	- Фамилия (доступно для поиска)
name	- Имя (доступно для поиска)
phone	- Телефон (доступно для поиска)
city	- Город (доступно для поиска)

В справочнике можно редактировать, удалять и добавлять записи.

Импорт.
Реализован механизм импорта нескольких записей из csv-файла
Формат файла:
строка заколовок (surname;name;phone;city)
разделитель ';'
тип csv

Экспорт.
Реализован механизм экспорта записей в csv-файла
Экспортировать можно весь справочник (только активные записи) или выбранные записи (результат поиска).
При экспорте создается csv-файл с наименованим по маске:
csv_phonebook_<дата выгрузки>_<время выгрузки>
